-----------------------ReadMe-----------------------
The program simulates a Pizza Company.
Customer can make requests on pizza customization by
adding cheese and toppings. When finished, customer
can choose to review receipt of pizza by checking out.
Checkout is only allowed for a pizza with at least one 
topping and one cheese.

The program is designed using decorator design pattern
where toppings and cheese are added using ingredeient 
decorators to base pizza.
